☁️ Google Drive Sync

Save your BuildLedger data to Google Drive — backed up and accessible from any device.

Step 1 — Google Cloud Console

Go to console.cloud.google.com → Create/select a project → Enable Google Drive API → Enable Google Picker API.

Step 2 — Create Credentials

Go to Credentials → Create API Key → Also create an OAuth 2.0 Client ID (Web application type). Add the URL where you open this HTML file as an Authorised JavaScript Origin (e.g. http://localhost or file:// may not work — host on a simple local server or GitHub Pages).

Step 3 — Enter your credentials
✅ Connected! Data auto-saves to BuildLedger_data.json in your Google Drive.
