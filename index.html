⚠️ Firebase Permission Error

Your Firestore security rules don't allow writing to the docs collection yet. Files are saved locally for now — fix this once and they'll sync forever.

Fix — Update your Firestore Rules

Go to console.firebase.google.com → your project → Firestore Database → Rules tab and replace with:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /ledger/{docId} { allow read, write: if true; } match /docs/{docId} { allow read, write: if true; } match /{document=**} { allow read, write: if false; } } }

Then click Publish. After that, re-upload your files and they'll sync across all devices.
